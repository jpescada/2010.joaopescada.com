Hiscox Installation

An interactive installation for Hiscox's stand at BIBA 2011 conference.
Built in Flash, used an XBox Kinect as the interface to control, by gestures, the flow of the presentation.

Roles: Ideation, Creative Tech Direction, Front-end Coding.
Languages: Actionscript, XML.
